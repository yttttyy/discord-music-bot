# Discord Music Bot

Музыкальный бот для Discord с воспроизведением из **YouTube** и **Spotify**, очередью и базовыми командами управления.

## Возможности

- Воспроизведение по ссылке (YouTube видео/плейлист, Spotify трек/плейлист/альбом) или по названию
- Умный поиск: находит именно музыку (до 7 минут, приоритет официального аудио «- Topic», отсев летсплеев/каверов/стримов по названию)
- Очередь треков с автопереходом, перемешивание, перестановка, прыжок к треку, страницы
- Бесконечное радио: очередь сама пополняется похожими треками (YouTube Mix), пока не выключишь
- Кнопки управления (пауза/скип/стоп) под «Сейчас играет» — отключаются в `!settings`
- Прогресс трека в `nowplaying`/`queue`, статус бота показывает текущий трек
- Повтор трека (loop)
- Плавные переходы между треками (fade-in/fade-out через ffmpeg)
- Минималистичные embed-сообщения (без эмодзи)
- Русские алиасы у всех команд
- Управление (skip/stop/pause/…) — только для тех, кто в голосовом канале с ботом
- Пустой войс: пауза сразу, выход через 5 минут; пустая очередь — выход через 10 минут

> Spotify напрямую не стримится. Бот берёт название трека из Spotify (через публичный embed — **без API-ключей и без Premium**) и находит эквивалент на YouTube.

## Команды

Префикс по умолчанию `!` (меняется в `.env`).

| Команда | Алиасы | По-русски | Описание |
|---|---|---|---|
| `!play <ссылка/название>` | `p` | `!плей`, `!п` | Воспроизвести или добавить в очередь |
| `!radio <ссылка/название>` | | `!радио`, `!р` | Бесконечное радио из похожих треков; без аргументов — выключить |
| `!skip` | `s`, `next`, `c` | `!скип`, `!с` | Пропустить текущий трек |
| `!skipto <номер>` | `jump` | `!прыг` | Пропустить всё до трека N и сыграть его |
| `!queue [страница]` | `list`, `q` | `!лист`, `!й`, `!очередь` | Показать очередь (по 10 на страницу) |
| `!shuffle` | `sh` | `!шафл`, `!ш` | Перемешать очередь |
| `!move <откуда> <куда>` | `mv` | `!переставить` | Переставить трек в очереди |
| `!clear [N]` | `clr` | `!клир`, `!кл` | Очистить очередь целиком или убрать N ближайших треков |
| `!remove <номер>` | `rm` | `!убрать` | Удалить из очереди один трек по его номеру |
| `!nowplaying` | `np`, `current` | `!чтощас`, `!нп`, `!щас` | Текущий трек с прогрессом |
| `!pause` | | `!пауз` | Пауза |
| `!resume` | `unpause` | | Продолжить |
| `!stop` | `leave`, `dc` | `!конец`, `!стоп`, `!к`, `!лив` | Остановить и выйти из канала |
| `!loop` | `repeat` | `!луп`, `!л` | Повтор текущего трека |
| `!settings` | | `!настройки` | Настройки сервера: `settings buttons on\|off` |
| `!help` | `h`, `commands` | `!хелп`, `!х`, `!команды` | Список команд |

> `!play` / `!плей` / `!п` **без аргументов** работает как переключатель пауза/продолжить: играет — поставит на паузу, на паузе — продолжит.

## Установка

1. Установка зависимостей:
   ```powershell
   npm install
   ```
2. Создать бота на https://discord.com/developers/applications, включить в разделе **Bot** опцию **MESSAGE CONTENT INTENT**.
3. Скопировать настройки:
   ```powershell
   Copy-Item .env.example .env
   ```
   Заполнить `DISCORD_TOKEN`. 
4. Пригласить бота на сервер с правами: `Connect`, `Speak`, `Send Messages`, `Read Message History`.
5. Запустить:
   ```powershell
   npm start
   ```

## Требования

- Node.js 18+
- FFmpeg в PATH
- **Deno** — нужен yt-dlp для решения JS-челленджа подписи YouTube (иначе многие видео отдают только картинки). Установка: `irm https://deno.land/install.ps1 | iex`. Бот сам подхватывает Deno из `~/.deno/bin`.
- YouTube-куки в `.env` (`YT_COOKIES_FROM_BROWSER=firefox`) — только для видео 18+

## Структура

```
src/
  index.js      — точка входа, обработчик сообщений
  config.js     — чтение .env
  sources.js    — резолв ссылок YouTube/Spotify + стриминг (yt-dlp)
  spotify.js    — метаданные Spotify через публичный embed
  queue.js      — очередь и аудиоплеер на сервер (радио-долив, авто-пауза)
  embeds.js     — фабрики embed-сообщений и кнопок (единый стиль ответов)
  settings.js   — пер-серверные настройки (data/settings.json)
  utils.js      — общие проверки (голосовой канал)
  commands/     — по файлу на команду
```
