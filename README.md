# 🎵 Discord Music Bot

Музыкальный бот для Discord с воспроизведением из **YouTube** и **Spotify**, очередью и базовыми командами управления.

## Возможности

- Воспроизведение по ссылке (YouTube видео/плейлист, Spotify трек/плейлист/альбом) или по названию
- Очередь треков с автопереходом
- Пропуск, пауза, продолжение, остановка
- Просмотр очереди и текущего трека
- Повтор трека (loop)
- Плавные переходы между треками (fade-in/fade-out через ffmpeg)
- Красивые embed-сообщения
- Управление (skip/stop/pause/…) — только для тех, кто в голосовом канале с ботом
- Авто-выход из канала, если очередь пуста 10 минут

> Spotify напрямую не стримится (это технически невозможно). Бот берёт название трека из Spotify (через публичный embed — **без API-ключей и без Premium**) и находит эквивалент на YouTube.

## Команды

Префикс по умолчанию `!` (меняется в `.env`).

| Команда | Алиасы | Описание |
|---|---|---|
| `!play <ссылка/название>` | `p` | Воспроизвести или добавить в очередь |
| `!skip` | `s`, `next` | Пропустить текущий трек |
| `!queue` | `list`, `q` | Показать очередь |
| `!clear [N]` | `clr` | Очистить очередь целиком или убрать N ближайших треков |
| `!remove <номер>` | `rm` | Удалить из очереди один трек по его номеру |
| `!nowplaying` | `np`, `current` | Текущий трек |
| `!pause` | | Пауза |
| `!resume` | `unpause` | Продолжить |
| `!stop` | `leave`, `dc` | Остановить и выйти из канала |
| `!loop` | `repeat` | Повтор текущего трека |
| `!help` | `h`, `commands` | Список команд |

## Установка

1. Установи зависимости:
   ```powershell
   npm install
   ```
2. Создай бота на https://discord.com/developers/applications, включи в разделе **Bot** опцию **MESSAGE CONTENT INTENT**.
3. Скопируй настройки:
   ```powershell
   Copy-Item .env.example .env
   ```
   Заполни `DISCORD_TOKEN`. Для Spotify ничего настраивать не нужно.
4. Пригласи бота на сервер с правами: `Connect`, `Speak`, `Send Messages`, `Read Message History`.
5. Запусти:
   ```powershell
   npm start
   ```

## Требования

- Node.js 18+
- FFmpeg в PATH
- **Deno** — нужен yt-dlp для решения JS-челленджа подписи YouTube (иначе многие видео отдают только картинки). Установка: `irm https://deno.land/install.ps1 | iex`. Бот сам подхватывает Deno из `~/.deno/bin`.
- YouTube-куки в `.env` (`YT_COOKIES_FROM_BROWSER=firefox`) — только для видео 18+

## Структура

```
src/
  index.js      — точка входа, обработчик сообщений
  config.js     — чтение .env
  sources.js    — резолв ссылок YouTube/Spotify + стриминг (yt-dlp)
  spotify.js    — метаданные Spotify через публичный embed
  queue.js      — очередь и аудиоплеер на сервер
  embeds.js     — фабрики embed-сообщений (единый стиль ответов)
  utils.js      — общие проверки (голосовой канал)
  commands/     — по файлу на команду
```
